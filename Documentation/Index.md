# Extension: Anatolkin Restaurant (anatolkin_restaurant)

## 1. Архитектура системы
Система построена на TYPO3 v13 с использованием строгого DI (Dependency Injection) и Autowiring.

### Модули и сущности:
* **Location**: Базовая сущность ресторана. Содержит адрес, часы работы и настройки интеграций.
* **Category**: Группировка блюд (Закуски, Пицца и т.д.).
* **Item**: Карточка блюда с ценой, изображениями и атрибутами.
* **Department**: Подразделение (Кухня, Бар) для управления графиком работы.

## 2. Плагины (Plugin Responsibilities)
| Плагин | Назначение |
| :--- | :--- |
| **Menu List** | Отображение сетки или списка блюд. |
| **Category Nav** | Навигация/якоря по категориям меню. |
| **Item Details** | Страница детального описания блюда. |
| **Checkout** | Интерфейс корзины и оформления заказа. |
| **Locations** | Список или карточка ресторанов сети. |
| **Integrations** | Шлюз для внешних виджетов (Booking, Delivery). |

## 3. Правила разработки
* **Полные пути**: Все правки осуществляются по абсолютным путям.
* **Full Overwrite**: Никаких "добавь строчку" — только полная замена содержимого файла.
* **Autowiring**: Использование конструкторов для внедрения зависимостей.
* **v13 Strict**: Возврат `ResponseInterface` во всех экшенах.
